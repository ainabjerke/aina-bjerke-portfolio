NUF test

The NUF Test helps pick the best design ideas.

  • New: Is it original and exciting?
  • Useful: Does it solve a real problem for users?
  • Feasible: Can we actually build it with our resources?
This method makes sure that ideas (feature/functionality) that have been selected are necessary, important, crucial to the core target audience of the app needs and eliminating ... features that are unnecessary elements that are not relevant and needed. To be able to select and pick feature and functionality that are most important persona and empathy map was being used to help prioritize essential features
